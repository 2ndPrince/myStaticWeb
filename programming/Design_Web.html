This web page is created using HTML and CSS only. Definitely, it is not the easiest way to create a website. But, it is the fastest way to learn and practice HTML and CSS coding techniques. Since this is a personal blog created mostly for learning purposes, this homepage does its job and I recommend you to make your own if interested in web techniques.

Several drawbacks exist on this manual web creation. It requires more works to handle various user web environment. For example, mobile users may see broken formatting on this website. Some web browers may also have formatting issues. Using an open source web creation tool such as Wordpress can solve these issues easily. 25% of all sites over the web was created on WordPress for its user easiness.

One important concept on the web is the relationship between client and server.
We are the client who request information to the server thru web browers.
Server is the host who responds client requests back thru web browers.
Note that web browers are the median between client and server.

Two main client web techniques are HTML and CSS. HTML is the main framework of web pages. It includes link that client can click on brower leading to another page. It also includes font, color, and graphics. CSS is mainly for artwork of web pages. It includes color, layout, figures and anything that helps client to request info by art style.

When asked, server finds the right information stored in its database and provides the info to the client. It is the matter of web security that the info is securely handed to the right client and no one else and with no data corruption. Java and PHP are the main server side programming languages that helps server to effectively respond requests from client.
