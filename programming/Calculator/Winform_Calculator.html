Why calculator?

It is a great example to get familiar with Visual Studio and its Winform platform. All you need are basic programming knowledges such as making functions and calling variables. Once you get used to the drag and drop interface & on-click trigger, VS takes care of anything else.

Click to download Source File

Download the Visual Studio and create a Visual C# project for Windows Forms Application.

On Form1.cs [Design], Design the interface using the toolbox on left menu. Button and Textbox are all we need.

Click button. Go to its properties. Each button has name that we can call, and values assigned. 0 to button "0", + to button "+" On button click, its value can be handled in C# to form a program. (here calculator)

Trouble Shootings

  • Button value is string. Needs conversion to double for mathematic operations -> Double.Parse, ToString()
  • Clicking operator twice cracks the program -> Handles as exception
  • Prevent first digit 0 numbers such as 090 or 0053 -> Handles as exception
